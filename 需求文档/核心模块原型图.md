# 核心模块原型图

**目标用户**：公司高层、部门负责人



**核心价值**：一屏掌握公司整体经营脉搏，快速发现亮点与风险。



```Plain Text
+--------------------------------------------------------------------------------------------------+
| [系统LOGO] 理财业务智慧决策平台                                        [搜索框: 搜产品/机构...] [用户: 西总 ∨] |
```

**原型说明**：

- **顶层KPI**：最核心的经营数据放在最上方，一目了然。

- **模块化布局**：每个分析维度是一个独立的卡片，清晰直观。

- **交互性**：规模结构图支持切换维度；地图支持悬浮提示；预警看板支持点击跳转。

- **数据溯源（后台逻辑）**：每个卡片的数据都明确指向了V4\.0规划中的数据表，例如“核心风险收益指标”的数据直接来自 `lh_dc_wealth_yield_weighted_average_table` 等预计算结果表。

### **原型二：360°产品档案**



**目标用户**：产品经理、投研人员、一线理财经理



**核心价值**：提供任何一款产品（无论是我方还是竞品）最全面的信息视图，是研究、对比、营销的基础。



```Plain Text
+--------------------------------------------------------------------------------------------------+

+--------------------------------------------------------------------------------------------------+
|                                                                                                  |

|                                                                                                  |
|  +-----------------------------------------------------------------------------------------------+

|  +-----------------------------------------------------------------------------------------------+
|  |                                                                                               |
|  | [业绩与风险] (当前选中标签页)                                                                 |

|  |  +-------------------+---------------+-----------------+------------------+-----------------+ |
```

**原型说明**：

- **一页总览**：页面顶部是产品的“名片”，包含最关键的实时信息和标签。

- **Tab式导航**：将海量信息归类到不同的标签页中，保持页面整洁。用户可以按需查看。

    - **基本信息**：来自 `lh_dc_wealth_product`。

    - **业绩与风险**：来自 `lh_dc_wealth_annualized...`, `lh_dc_wealth_max_drawdown...` 等指标表。

    - **资产配置**：来自 `lh_dc_wealth_asset_penetrating_prop` 等表，可用饼图展示。

    - **费率与规则**：来自 `lh_dc_wealth_commision_charge` \& `lh_dc_wealth_purchase`。

    - **营销亮点 \(特色功能\)**：自动从指标表中抓取该产品表现优异的维度，如“近1年夏普比率在同类固收\+产品中排名前5%”。

- **可视化**：净值走势图是最核心的可视化元素，必须清晰、可交互。

好的，西总。我们继续，将V4\.0蓝图中剩余的所有模块都转化为详细的线框原型。这将确保我们对整个系统的每个角落都有清晰、统一的认知。



我将按照业务域的顺序，逐一绘制以下模块的原型：



- **产品与投研中心** \(深化模块\)

    - 原型四：资产配置透视

    - 原型五：市场与投研洞察 \(含投资经理画像\)

- **营销与客户中心**

    - 原型六：渠道与区域分析

    - 原型七：自动化营销支持 \(产品对比 \& 亮点挖掘\)

- **智能风控中心**

    - 原型八：风险事件预警

    - 原型九：产品健康度扫描

**核心价值**：从宏观市场和微观管理人两个层面，提供竞品对标和投研能力评估的依据。

- **销售区域分析**：切换到“销售区域分析”Tab后，会展示与“公司经营驾驶舱”中类似的销售热力图，但可以下钻到市级，并列出该区域销量Top 10的产品和渠道。

### **原型七：自动化营销支持**



**目标用户**：一线理财经理、市场内容运营

```Plain Text
+--------------------------------------------------------------------------------------------------+
|                                                                                                  |
|  [产品PK台] [营销亮点一键生成]                                                                   |
|  (当前选中: 产品PK台)                                                                            |
|                                                                                                  |
|  <---------------------------------------------------------------------------------------------->|
|  |  产品1: [ 我司王牌产品A ]  vs  产品2: [ 竞品爆款X ]  vs  产品3: [ (可选) ]  [开始对比]          |
|  <---------------------------------------------------------------------------------------------->|
```
