# 联合智评数据应用平台 — 系统设计文档

**日期**：2026-06-10
**版本**：V1.0
**我司**：中邮理财

---

## 1. 项目背景

基于"联合智评"数据结构（80张已采购PostgreSQL表），构建中邮理财一体化数据应用平台。面向运营部、投资部、风险部、市场部，提供产品驾驶舱、营销策略、智能风控、投研支持四大核心能力。

## 2. 技术栈

| 层级 | 选型 |
|------|------|
| 后端框架 | Flask 3.x + Blueprint 模块化 |
| ORM | SQLAlchemy 2.x（反射模式，自动映射已有表） |
| 数据库驱动 | psycopg2-binary（远程PG）+ SQLite（本地用户库） |
| 用户认证 | Flask-Login + werkzeug 密码哈希 |
| 前端 | Bootstrap 5 (CDN) + ECharts 5 (CDN) + Jinja2 模板 |
| Python | 3.14 |
| 部署 | Flask 开发服务器 / gunicorn（生产） |

## 3. 系统架构

```
Flask App
├── Blueprint: auth            # 用户认证（登录/登出/修改密码）
├── Blueprint: dashboard       # 公司经营驾驶舱
├── Blueprint: product         # 产品与投研中心
├── Blueprint: marketing       # 营销与客户中心
├── Blueprint: riskctrl        # 智能风控中心

├── Blueprint: api_marketing   # 营销数据API
├── Blueprint: api_risk        # 风控数据API
└── 共享层
    ├── models/                # 反射表定义 + User模型(SQLite)
    └── services/              # 业务查询逻辑
```

## 5. 数据层设计

### 5.1 设计策略

- **反射优先**：80张表已在PostgreSQL中建好，用 `Table(..., autoload_with=engine)` 自动映射
- **服务层封装**：每个业务域一个Service类，SQL逻辑集中管理
- **年份分表**：17张 `_20192025` 分表用动态表名函数处理

### 5.2 Model 分组

```
app/models/
├── base.py          # 反射引擎 get_engine(), reflect_table()
├── product.py       # lh_dc_wealth_product, lh_dc_wealth_sub_product, lh_dc_wealth_series
├── indicator.py     # 17张指标计算表 + 16张指标统计表 + 年份分表工具
├── asset.py         # 底层资产、资产占比、穿透详情、投资范围
├── net_worth.py     # 净值历史、初始净值、理财网渠道净值
├── org.py           # 机构、代销机构、签约代销、区域
├── manager.py       # 投资经理、产品经理关联
├── trade.py         # 申购规则、费率、浮动费率、兑付历史
└── risk.py          # 破净统计、负收益、杠杆率、资金站岗
```

### 5.3 Service 分组

```
app/services/
├── cockpit_svc.py    # 驾驶舱KPI、规模结构、预警摘要
├── product_svc.py    # 产品详情、净值查询、排行榜
├── asset_svc.py      # 资产配置对比、穿透持仓
├── research_svc.py   # 投资经理画像、机构动态
├── channel_svc.py    # 渠道效能、区域分析
├── marketing_svc.py  # 产品PK对比
├── alert_svc.py      # 预警规则、预警列表
└── health_svc.py     # 产品健康扫描
```

| 三 | 夏普/卡玛/回撤/收益率分表, `classification`, `wealth_org` |

## 8. 中邮理财特殊处理

- 产品排行、对比中，中邮理财产品行高亮（`table-primary` class）
- 渠道分析默认筛选中邮理财作发行方
- 竞品分析中，中邮理财作为对标基准
- 驾驶舱标题区显示"中邮理财"品牌标识
